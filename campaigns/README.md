## Estrutura por Campanha

```
campaigns/nome-da-campanha/
├── core/           # Master plot, temas, nemesis
├── npcs/
├── dungeons/
├── sessions/
├── factions/
├── monsters/
├── encounters/
├── handouts/
├── players/
└── extra-lore/
```

Ver `.cursor/rules/campaign-md.mdc` para convenções detalhadas.
