# Campaign Core Files

This directory contains the core structural elements of the Black Hollow campaign, organized following "The Complete Guide to Creating Epic Campaigns" methodology.

## File Structure

### [00-master-plot.md](00-master-plot.md)

"The Sentence" - The fundamental engine of the story that drives the campaign.

### [01-themes.md](01-themes.md)

The three main themes that shape the campaign's narrative:

- Power vs. Knowledge
- Ambition's Price
- Order vs. Chaos

### [02-nemesis.md](02-nemesis.md)

The Gravedigger's complete profile:

- Multiple layers of deception
- Goals and motivations
- Methods and resources
- Physical and emotional characteristics
- Development arc
- Early game presentation

### [03-master-plot.md](03-master-plot.md)

The detailed plot structure following the four stages:

1. Obtaining Requirements
2. Building & Preparation
3. Testing Phase
4. The Final Goal

Includes timeline pressure, complications, and adaptation points.

### [XX-campaign-structure.md](XX-campaign-structure.md)

Pre-guide campaign outline and reference:

- Level progression notes
- Adventure milestones
- Reward tracking
- Current campaign status

## Using These Files

Follow this order when planning and running sessions:

1. **Master Plot (00)** - Review the core conflict
2. **Themes (01)** - Apply thematic elements
3. **Nemesis (02)** - Plan antagonist actions
4. **Plot Structure (03)** - Track progress and adapt
5. **Campaign Structure (XX)** - Reference current status

## Relationship to Other Directories

- `campaign/npcs/` - Individual NPC details
- `campaign/dungeons/` - Location-specific content
- `worldbuilding/` - Broader world context

## Maintenance

- Update [03-master-plot.md] as the campaign progresses
- Use [XX-campaign-structure.md] to track completed milestones
- Keep [02-nemesis.md] current with player discoveries
- Reference [01-themes.md] when creating new content
