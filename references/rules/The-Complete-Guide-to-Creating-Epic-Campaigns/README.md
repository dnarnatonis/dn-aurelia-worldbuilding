# The Complete Guide to Creating Epic Campaigns

This directory contains the complete extraction and organization of "The Complete Guide to Creating Epic Campaigns" by Guy Sclanders from the How to be a Great GM YouTube channel.

## Book Structure

The book has been organized into sequential markdown files for easy navigation:

### Core Chapters (Complete)

- **00-introduction.md** - Author's introduction and philosophy
- **01-where-to-start.md** - Master Plot formula and Lake Metaphor
- **02-nemesis-theme-genre.md** - Nemesis types, Theme selection, and Genre considerations
- **03-plotting.md** - 4-stage nemesis planning, adventure types, and 121/122 methods
- **04-designing-adventures.md** - Adventure creation, player types, and practical examples
- **05-adventures-on-the-fly.md** - Improvisation techniques and plot twists
- **06-character-plots.md** - Character backstory adventures and player-driven content
- **07-timing.md** - Session pacing and timing management
- **08-seeding.md** - Foreshadowing techniques and seed management
- **09-npcs.md** - NPC creation, types, and personality factors
- **10-conclusion.md** - Final thoughts and post-game feedback

### Additional Chapters (Complete)

- **09b-npc-dialogue.md** - Comprehensive guide to NPC dialogue creation
- **11-beginning.md** - Campaign opening techniques and methods  
- **12-ending.md** - Campaign conclusion strategies and the "5 C's"

## About This Book

This is a comprehensive guide to campaign creation and management. The book is structured to take you through the entire process of creating and running a campaign, from initial concept to conclusion.

## How to Use This Directory

Each chapter is organized in its own directory with a main file (00-main.md) and additional files for subsections. The page numbers from the original book are preserved in the file headers for reference.
