# Myths — Lentes sobre a History

Lendas, histórias de criação e narrativas mitológicas. *O que as pessoas sabem* — ou acham que sabem.

Diferem da **history** (fatos) porque podem ser verdadeiras, falsas ou contradizer umas às outras.

## Organização por Era

```
myths/
├── 01-origins/
├── 02-mythical-age/
├── 03-age-of-balance/
├── 04-age-of-heroes/
├── 05-age-of-man/
└── (deidades standalone: Veshka, Ulmathar, etc.)
```

## Classificação de Penetração (P1–P5)

| Nível | Quem conhece |
|-------|--------------|
| P1 | Todo mundo sabe |
| P2 | Maioria das pessoas |
| P3 | Grupos específicos |
| P4 | Poucos guardiões |
| P5 | Ninguém sabe |
