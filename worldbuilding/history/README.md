# History — Fatos do Mundo

Aqui ficam os **fatos** que servem de framework de consistência para Aurelia.

- Coisas que provavelmente nunca usaremos diretamente em sessões
- Base para que culture e myths façam sentido
- Linha do tempo, eventos, geografia factual
- Não confundir com o que as pessoas *acham* que aconteceu (isso vai em myths)
