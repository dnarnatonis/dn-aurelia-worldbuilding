# Myths — Lentes sobre a History

Os mitos são *o que as pessoas sabem* sobre o mundo — ou acham que sabem. Diferem da **history** (fatos) porque:

- Podem ser verdadeiros, falsos ou parciais
- Podem contradizer uns aos outros
- Variam conforme a penetração (quem conhece)

## Classificação de Penetração (5 níveis)

| Nível | Código | Quem conhece |
|-------|--------|--------------|
| 1 | `P1` | Todo mundo sabe |
| 2 | `P2` | Maioria das pessoas |
| 3 | `P3` | Grupos específicos (eruditos, clérigos, nobres) |
| 4 | `P4` | Poucos guardiões, círculos fechados |
| 5 | `P5` | Ninguém sabe (perdido/esquecido) |

## Template para novo mito

```markdown
---
penetration: P2
veracity: unknown
contradicts: []
tags: [myth, religion, origin]
---

# Nome do Mito

## O que se diz

Descrição do mito como as pessoas o contam.

## Penetração

P2 — conhecido pela maioria.

## Notas do Narrador

[Opcional] Verdade por trás do mito, se diferente.
```
